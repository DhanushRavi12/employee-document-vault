# Employee Document Vault

## Overview
A secure document management system for employees.

## AWS Services Used
- Amazon S3
- Amazon DynamoDB
- Amazon Cognito
- AWS Lambda
- Amazon API Gateway
- AWS IAM

## Features
- Upload Documents
- View Documents
- Delete Documents
- Role Based Access Control

## API Endpoints

GET /files

POST /upload

DELETE /files/{id}
